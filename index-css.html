There are 3 ways we can add CSS to our HTML files

  1. Inline with the style HTML attribute

  2. Internal with the ≶style> element

  3. External.css file

  1. Inline using style HTML attribute: adding "style" attribute dirently in the HTML file next to other tags.

                ≶body>
            ≶h1 style="font-size: X px;"> Heading 1
            ≶/h1>

            ≶h1 style="color: blue; font size: X px"> Heading 2
            ≶/h1>

  2. Internal with the ≶style> element: Adding a seperate tag named 'style'

                ≶style>
                h1
                {
                    background color: blue;
                    border: X px;
                    padding: Y px;
                }
            ≶/style>

  3. External.css file:(prefferred) Making a seperate CSS file and then linking it to the HTML file which it is formatting.

    (I am using this way.)

                ≶link rel="stylesheet" href="(Link to the CSS file)">

  3. Display types: Inline, inline-block, block etc.

    4. This helps control layouts of the page.

    For example: consider the following code:
    HTML:
                    
                ≶div class="display1" > This is inline ≶/div>

    CSS:
                    .display1
                    {
                        display: inline;
                    }

                .display2
                    {

                    }

                .display3
                    {

                    }

    This is inline
    
                
    This is inline-block: allows padding for each element
    
                
    This is block:takes full width of the screen
