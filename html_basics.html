  1. Linking one page to another:

    To link one page to another in html, we use the syntax/tag: <!a href="(insert link here)">(Name of link) <!/a>
    Note: To open a link in a new tab: <a href="(Link)"> target="_blank">

  2. Adding images:

    To add images, use syntax: ≶img src="(Name of the image)".(image file type) width= "X" height="Y"> 
                    To link an image to another page, use syntax:
                ≶a href="(page link)">
                ≶img 
                    src="(image).jpg"
                    width="X"
                    height="Y"
                    alt="Link" 
                    (alt tag shows the message it contains when the image does not load)
