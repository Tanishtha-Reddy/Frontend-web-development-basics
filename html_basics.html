  1. Adding images:

    To add images, use syntax: 
                    ≶img src="(Name of the image)".(image file type) width= "X" height="Y">
    To link an image to another page, use syntax: 
                    ≶a href="(page link)">
                ≶img 
                    src="(image).jpg"
                    width="X"
                    height="Y"
                    alt="Link" 
                    (alt tag shows the message it contains when the image does not load)

  2. Favicons

                    ≶ link rel="Shortcut icon" href="(Name).png" type="image".png >

  3. Tables

                ≶table border="X" width="100%">
            ≶tr>
                >td> Cell1 >/td>
                >td> Cell2 >/td>
                >/tr>
                    >td>Cell2>/td>
                    >td>Cell2>/td>
                >/tr>
            >/table>

    Output

    Cell1 Cell2
    Cell2 Cell2

  4. Forms: This is how we send data to a server

                    ≶ form 
                method= "GET" action="somepage.html" >

                ≶ button type = "submit" > Submit form ≶/button >

                ≶ /form >

    Output

  8. Embedding Videos: To add videos from YouTube or elsewhere directly into your website

                        ≶ iframe width="X" height="Y" src="(link of the video) frameborder=Z" allowfullscreen >
                    ≶ /iframe >

    Output

  9. Linking one page to another:

    To link one page to another in html, we use the syntax/tag: <a href="(insert link here)">(Name of link) <!/a>
    Note: To open a link in a new tab: 
                <a href="(Link)"> target="_blank">
